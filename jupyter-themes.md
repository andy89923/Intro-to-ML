## Jupyter-themes
Reference: [jupyter-themes Github](https://github.com/dunovank/jupyter-themes)
```
# install jupyterthemes
$ pip3 install jupyterthemes

# upgrade to latest version
$ pip3 install --upgrade jupyterthemes
```

```
$ jt -l # show all the themes
$ jt -t [Theme name]
$ jt -r # reset

# My settings
$ jt -t monokai -f roboto -fs 11 -cellw 95% -nfs 11 -tfs 11 -ofs 10 -dfs 10 -T -N
```
